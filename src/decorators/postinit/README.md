# Dataclass Post Init

Exploring `__post_init__` for dataclass validation.

1. `legacy_class_example.py` - Traditional class with manual `__init__` and `__repr__` 
2. `basic_dataclass_example.py` - Basic dataclass with automatic methods
3. `dataclass_post_init_demo.py` - Dataclass with `__post_init__` for validation and normalization 
4. `item_price_validator.py` - Price validation using `__post_init__` 
5. `staff_email_generator.py` - Auto-generated email addresses
6. `contact_phone_cleaner.py` - Phone number normalization
7. `purchase_order_demo.py` - Timestamp generation with validation
8. `immutable_settings.py` - Frozen dataclass with validation

All scripts are standalone examples demonstrating different aspects of Python dataclasses and the
`__post_init__` hook, using only the standard library.
